Scenarios & Personas
A large language model generates 3 distinct investor personas for each of the 8 investor types. Your scenario name, description, and any shocks you define are passed as context — configure those tabs first for more scenario-relevant profiles.

The model outputs relative mispricing signals — where agent behaviour collectively pushes sector prices above or below a neutral consensus baseline. This is not a price forecast; consistent directional signals across runs are investment theses to investigate.

Presets

Agents cannot short-sell — bearish pressure is bounded by existing holdings. The model reflects long-only investor behaviour.

Select a preset above, or generate profiles for your own scenario using the button below.

Price sensitivity per sector

How strongly sector prices respond to buying and selling pressure. Higher values mean prices move more sharply for a given order imbalance. Defaults reflect real-world liquidity differences across sectors.

How strongly non-price signals — news, social media, analyst commentary — broadcast into the market and influence agent behaviour. Scale 0–1.
How quickly the shock narrative decays. After this many days, herding intensity halves — agents diverge and two-sided flow resumes. Shorter for sharp one-off events (5); longer for sustained macro narratives (20–30).

Investor mix
Controls the number of agents and capital weight for each investor type. Defaults are calibrated to realistic relative market weights.
Using defaults — no changes needed to run.

Adjust the number of agents or capital base for each investor type. Leave blank to use defaults (30 agents per type). Over- or under-weighting a type shifts the population composition and will affect which dynamics dominate.

Run configuration
Set the simulation duration and number of runs. Agent profiles not loaded — return to Scenarios & Personas.
Without a shock configured in the Shocks tab, prices will stay near 100 — the model is mean-reverting by design. For visible price dynamics, add at least one shock with the market channel enabled.
Each run uses a different combination of agent profiles. More runs = more robust results — but each run takes compute time. Start with 10 runs to check your setup before committing to 100+.

Final mispricing by sector
Ceteris paribus relative (mis)pricing pressure — the baseline of 100 is a proxy for prevailing market consensus, not a fixed price. A +5% reading means agents systematically priced that sector 5% above consensus; it does not mean the absolute price rose. A tight distribution away from zero is a robust finding. A wide spread means the result is sensitive to population composition. Near-zero with a tight distribution = strong signal of no mispricing. Each dot = one run. Box = P25–P75; whiskers = full range.
